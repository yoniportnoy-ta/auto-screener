1
Download the extension
Click the Download button above. You'll get a file like auto-screener-extension-{{ extensionVersion }}.zip.
Find it in your Downloads folder and double-click to unzip it. Keep the unzipped folder somewhere stable — Chrome will read from it directly, so don't move it after.
2
Open Chrome's extensions page
In Chrome's address bar, type chrome://extensions/ and press Enter.
(Browsers block clickable chrome:// links for security, so you have to paste it yourself.)
3
Turn on Developer mode
In the top-right of that page, flip the Developer mode toggle on. Three new buttons appear (Load unpacked, Pack extension, Update).
If you don't see the toggle, your Chrome is managed by IT (the toggle is hidden). Use Chrome Canary or ask IT to allow developer mode.
4
Load the unpacked extension
Click Load unpacked, then select the folder you unzipped in step 1. The Auto Screener tile should now appear in the list.
Click the puzzle-piece icon in Chrome's toolbar (top-right) → click the pin next to Auto Screener so it stays visible.
✓
You're done
